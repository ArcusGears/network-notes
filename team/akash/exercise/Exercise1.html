The following are the basic commandline inputs you can use to navigate through and use Git:

  • git init : creates a new git repository
  • git clone /path/to/repository : creates a working copy of a local repository by running the command
  • git add (filename) or git add * : can be used to propose changes
  • git commit -m "Commit message" : actually confirms and commits the changes
  • git push origin master : sends the changes to your remote repository. It is possible to select other branches by replacing the word master with the branch name.
  • git pull : updates your local repository to the newest commit
